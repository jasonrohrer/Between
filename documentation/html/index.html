Between

a video game by Jason Rohrer


NOTE:

In order to play Between, you need to edit the following settings file (in the settings folder):

settings/serverFinderURL.ini

You can edit this file with Notepad or TextEdit.

You will find a URL inside, but that URL no longer works correctly. Replace that URL with:

http://www.northcountrynotes.org/game7/serverLocation.txt

Save the file, and then launch Between. You should be able to connect to the server and play.
